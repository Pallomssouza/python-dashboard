# Logística de Transportes

Projeto desenvolvido para a disciplina de Logística de Transportes.

**Autora:** Palloma Rocha de Souza

## Descrição

Este projeto é um dashboard interativo desenvolvido em Python utilizando o Streamlit para análise de custos logísticos entre capitais brasileiras, considerando centros de distribuição, custos de transporte, estoque e custos fixos.


## Como Executar no Copilot

1. Instale as dependências principais (caso ainda não tenha feito):
	```bash
	pip install streamlit pandas numpy
	```

2. Execute o dashboard:
	```bash
	streamlit run app.py
	```
	O terminal mostrará um link para acessar o dashboard no navegador.

3. Se preferir, também pode usar o comando do makefile:
	```bash
	make start
	```

## Dependências Principais
